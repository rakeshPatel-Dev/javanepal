# Feature Ticket

Use this template when proposing or implementing a new feature.

## Metadata

- **Ticket ID**: `FT-{NNN}`
- **Status**: `proposed | in-progress | done | deferred`
- **Priority**: `low | medium | high | critical`
- **Target Release**: `v{MAJOR}.{MINOR}`

## Template

```markdown
### Problem
<!-- What need does this address? -->

### Proposed Solution
<!-- High-level approach -->

### Files Affected
<!-- Known touch points -->

### Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2

### Notes
<!-- Edge cases, open questions, references -->
```
